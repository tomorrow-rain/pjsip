# PJSIP 
PJSIP视频通话客户端（计算机网络实验课程）

## 环境和工具

1. 操作系统：Windows 10
2. IDE：Visual Studio 2017
3. PJSIP 2.9
4. Qt Creator 4.3.0

2. FFmpeg 4.2.1-win32

3. SDL2-devel-2.0.12-VC

## 使用方式

1. 完整的编译文件在pjsip文件夹下，可以按需修改

2. 直接运行chat文件夹下的chat.exe，即可进行视频通话

   点击chat.exe，允许防火墙

   输入对方ip，点击call，等待对方接听
   或等待对方呼叫，点击answer

   可以语音或语音+视频
